## 六、发现洞察

### 板块分化规律

```
算力基建新候选均值 YTD: +62.3%   ← 最强方向，几乎所有光/芯/封测都涨
模型平台新候选均值 YTD: +23.6%   ← 分化严重，端侧硬件 > AI软件
具身硬件新候选均值 YTD: -12.0%   ← 除了传感器，多数还在消化估值
应用服务新候选均值 YTD: -18.2%   ← 全线下跌，仅工业AI独苗正收益
```

### 关键发现

1. **光芯片/光器件是算力基建中的 Alpha 富矿**：源杰科技（+258%）和光迅科技（+195%）远超光模块均值（~+70%），说明产业链利润在向上游芯片/器件环节转移。

2. **端侧 SoC 中出现了新的"六小龙"**：北京君正（+119%）、星宸科技（+88%）、中科蓝讯（+52%）代表了比恒玄/瑞芯微更新一代的端侧 AI 芯片公司。

3. **具身硬件进入"零部件精品化"阶段**：传感器（奥比中光 +45%、华依科技 +11%）和核心传动（绿的谐波 +143%）是仅有的正收益方向，纯整车/整机概念股全线下跌。

4. **应用层只有工业 AI 能挣钱**：中控技术（+109%）是 AI 应用层唯一正收益标的，其余全线负收益——市场在说「应用元年」还为时过早。
