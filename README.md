# 2026 全球 AI 产业链利润分配 · 桑基图报告

> 单页可视化：以 **国家 → 公司** 两层节点，展示 2026 年全球 AI 产业链利润（美元）从地缘阵营向代表性企业流动的份额结构。

## 项目目标

- 聚焦「**利润分配**」这条单一信息线，用一张可交互的桑基图讲清楚：
  1. 第一层 —— **国家 / 区域阵营**（美、中、欧、日韩、其他）拿走多少利润；
  2. 第二层 —— **代表性公司**（NVIDIA、TSMC、OpenAI、字节、ASML …）从中切走多少。
- 所有数值标注为「2026 年估算（E）」，单位 **十亿美元（USD bn）**，源自公开卖方一致预期与产业研究的加权聚合，仅作研究示意。
- 报告强调「**视觉克制 + 信息密度**」：黑底 + 单色高对比 + 等宽数据字 + 极细节点标签，让桑基流量本身成为视觉主角。

## 设计取向

- **配色**：近黑底 `#0B0D10` + 单一暖橙 `#FF7A45` 高亮 + 中性灰阶。避免多彩渐变污染语义。
- **字体**：标题用 `Fraunces`（衬线，显示性格），正文 `Inter`，数据 / 标签 `JetBrains Mono`（等宽，对齐数字）。
- **签名元素**：左侧固定侧栏的「**Profit Ledger**」+ 时间戳印章 + 等宽表格，让页面读起来像一份内部研究备忘，而不是仪表盘。

## 交互

- 悬停链路：高亮当前节点对，显示金额、占比与所属公司 / 国家。
- 点击节点：聚焦其下游 / 上游，淡化其他链路。
- 顶部维度切换：可在「**按地区 → 按公司**」与「**按硬件层 → 按模型层**」两套切分之间切换。
- 键盘可达：节点可用 Tab 聚焦，Esc 取消聚焦。

## 数据

详见 [`docs/methodology.md`](docs/methodology.md)。核心结论（仅供讨论）：

- 美国阵营拿走约 **71%** 的产业链利润，其中 NVIDIA 一家约占 **23%**；
- 中国阵营约 **15%**，字节跳动 + 腾讯 + 阿里合计占其中近一半；
- 欧洲（ASML、ARM 等）与日韩（TSMC、三星、SK 海力士）合计约 **14%**，但 **ASML + TSMC 两家**贡献了欧洲 + 日韩利润的近六成。
